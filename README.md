# TradeFlowAnalysis
A data analytics project for learning vessel movement and trade flow analysis using vessel data from MarineTraffic.

# Project Plan
I have included my plan for this project in the subsection titled "Preliminary Plan". Due to my current inexperience with trade flow data--specifically AIS vessel data--this plan is subject to changes as I progress in this project.

## Preliminary Plan
1. Data sourcing (branch: DataSourcing)
I have so far explored using data from MarineTraffic. However, due to the datastream being out of my budget, I am currently exploring using WebSockets to stream vessel data from aisstream.io.

2. Data processing and analysis
I will explore using the OpenAIS toolkit in this stage of my project to process the raw data I obtain from aisstream.io.
